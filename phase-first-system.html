Valid setups
A1 — SECOND LEG RECLAIM
fresh impulse → krótka pauza / płytki dip → reclaim → druga fala
  • Token świeży, ruch dopiero się rozwija
  • Pierwszy impuls nie sprzedany od razu całkowicie
  • Dip płytki lub szybko odkupiony
  • Widoczny reclaim lokalnego poziomu na 1s
  • Wejście przed drugą/trzecią falą — nie po finalnym blow-offie
Przykłady: Pup · Bob · ELUN · Beekeeper · Jupiter · BTC
A2 — GRIND CONTINUATION
trend schodkowy → higher lows → płytki pullback → kontynuacja
  • Cena nie robi jednego pustego knota — buduje strukturę
  • Pullbacki kontrolowane, szybko odkupywane
  • Po wejściu nie trzeba czekać minutami
  • HH/HL wyraźne na wykresie
  • Narracja ma paliwo do kolejnej fali
Przykłady: For The Benefit · BTC · munad
Anti-patterny — 5 gwarancji straty
D1 — ONE-CANDLE PUMP CHASE
Jedna świeca robi większość ruchu → kupno po niej → dump
Jeśli nie byłeś przed tą świecą → HARD PASS dopóki nie powstanie nowa baza
Elon Pet Bee · ASS · XD News · Kali · Toe Licker
D2 — POST-BLOWOFF KNIFE CATCH
Pionowy pump → duża czerwona → brak reclaimu → kupno "bo tanio"
Pierwszy dump z topu bez reclaimu = nie dip, tylko początek śmierci
Darkraider · Charity · DAND · pumpfun Seal · Kali
D3 — LOWER HIGH AFTER TOP
Top → dump → słabe odbicie → lower high → entry
LH po blow-offie bez wybicia poprzedniego poziomu = dystrybucja, nie dip
D4 — AVERAGING WITHOUT RECLAIM
Entry sensowne → re-entry/averaging → brak reclaimu → większa strata
Drugie wejście musi mieć NOWY setup. Nie kontynuację starego pomysłu.
Banana (5B) · Rocket Frog (4B) · HERO (7B)
